StrictClock v1.3 brings a massive set of new features and stability improvements to enforce your alarms more strictly than ever.

### New Features & Challenges
- **Puzzle Challenges:** Added 4 distinct configurable wake-up challenges: Math, Flashbang, Camera Object Detection, and QR Code scanning.
- **Camera Challenge Options:** Expanded the object detection pool to include "Grass", "Cat", and "Dog" alongside several other objects.
- **Invincible Mode:** Implemented Device Administrator privileges to prevent uninstallation while an alarm is active.
- **Anti Power Off:** Implemented an Accessibility Service to prevent users from powering off the device when an alarm is ringing.
- **Flashbang Configuration:** Added a setting to dismiss the flashbang effect by double-tapping the screen, allowing users to focus purely on the puzzle.
- **Predictive Back Gesture:** Added a toggle in Customisation Settings to enable modern Android predictive back gestures when returning to the launcher.

### UI & UX Improvements
- **12/24 Hour Sync:** The main wake-up screen and dashboard clocks now accurately respect the device's system-level 12/24 hour settings.
- **Minimalist Dashboard:** Removed redundant menu icons from the Alarms and Wake-up screens for a cleaner look.

### Under the Hood
- R8 minification and resource shrinking are now fully enabled, bringing the total app footprint down to just ~3MB.
- Upgraded target SDK to 34.
